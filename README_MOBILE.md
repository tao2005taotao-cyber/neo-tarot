# Neo Tarot 移动设备使用指南

## 应用特点
Neo Tarot 已经针对移动设备进行了全面适配，支持：
- 触屏操作（滑动转动、下拉抓取/揭示）
- 响应式布局（自动适配不同屏幕尺寸）
- 移动友好的交互体验

## 使用方式

### 方法一：在线体验

1. **访问在线版本**：
   - 主镜像：https://neotarot.vercel.app/（全球用户推荐，需魔法上网）
   - 备用镜像：https://hqzzdsda.github.io/neo.tarot/（国内网络兼容性较好）

2. **移动设备操作指南**：
   - **滑动**：在屏幕上左右滑动来转动牌阵
   - **下拉**：按住中轴卡牌向下拉动来抓取
   - **释放**：下拉超过阈值后释放，揭示牌面

### 方法二：本地运行（推荐）

由于浏览器安全限制，摄像头功能需要在 HTTPS 或 localhost 环境下才能正常使用。如果要在本地运行：

#### 1. 准备工作
- 确保您的电脑和移动设备连接在同一无线网络
- 下载并安装 Python（如果没有的话）

#### 2. 启动本地服务器

```bash
# 1. 克隆仓库
git clone https://github.com/hqzzdsda/neo-tarot.git

# 2. 进入目录
cd neo-tarot

# 3. 启动本地服务器
python -m http.server 8000
```

#### 3. 移动设备访问

1. 在电脑上查看本地 IP 地址：
   - Windows：按 `Win + R`，输入 `cmd`，然后运行 `ipconfig`，查找 "IPv4 地址"
   - Mac：打开终端，运行 `ifconfig | grep inet`，查找本地 IP

2. 在移动设备浏览器中输入：`http://您的本地IP:8000`（例如：http://192.168.1.100:8000）

## 功能说明

### 手机/平板专属功能
- **自动切换交互模式**：移动设备自动使用触屏控制
- **优化的视觉效果**：根据屏幕方向调整卡牌布局
- **触屏友好的按钮**：增大点击区域，方便触屏操作

### 常见问题

1. **摄像头无法使用**
   - 确保在 HTTPS 或 localhost 环境下运行
   - 允许浏览器访问摄像头权限

2. **手势不灵敏**
   - 确保光线充足
   - 保持手部在摄像头范围内
   - 尝试调整摄像头位置

3. **页面显示异常**
   - 刷新页面
   - 确保浏览器支持 WebGL（现代浏览器都支持）
